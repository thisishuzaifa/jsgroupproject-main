This page is about localStorage

Local storage is a read-only propery of the widnow interface which allows you to access the storage in the browser to be used across borwser sessions.

This creates a storage object that can be used to store data in the browser. Local storage uses key value pairs to store access and modify data.

myStorage = window.localStorage;

you can add a data item like this:

myStorage.setItem("key", "value");

you can read the stored item like this:

myStorage.getItem("key");

you can remove the item like this:

myStorage.removeItem("key");
